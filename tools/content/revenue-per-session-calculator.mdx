Use this calculator to estimate revenue per session.

## What this calculator does

The revenue per session calculator helps measure how much revenue each website session generates on average.

It is useful for:

- ecommerce analysis
- landing page review
- traffic monetisation benchmarking
- performance reporting

## Formula

**Revenue per Session = Total Revenue ÷ Sessions**

Where:

- **Total Revenue** = revenue generated in the period
- **Sessions** = total website sessions in the same period
- **Revenue per Session** = average revenue generated per session

## Example calculation

If:

- Total revenue = **85000**
- Sessions = **12000**

Then:

- Revenue per session = **85000 ÷ 12000**
- Revenue per session = **7.08**

## What is revenue per session?

Revenue per session measures the average revenue generated from each website visit.

## Common mistakes

Common mistakes include:

- using users instead of sessions without noting the difference
- comparing inconsistent periods
- including non-revenue traffic without context
- assuming a high traffic volume always means strong monetisation

## Revenue per session vs average order value

These are closely related.

- **Revenue per session** measures revenue per visit
- **Average order value** measures revenue per order
