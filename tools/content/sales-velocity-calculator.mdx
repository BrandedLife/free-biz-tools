## What this calculator does

The sales velocity calculator helps estimate how much revenue moves through your sales pipeline each day.

It is useful for:

- pipeline analysis
- sales forecasting
- CRM review
- team performance benchmarking

## Formula

**Sales Velocity = Opportunities × Average Deal Value × Win Rate ÷ Sales Cycle Length**

Where:

- **Opportunities** = number of active sales opportunities
- **Average Deal Value** = average value of each opportunity
- **Win Rate** = percentage of opportunities won
- **Sales Cycle Length** = average number of days to close
- **Sales Velocity** = expected revenue moving through the pipeline per day

## Example calculation

If:

- Opportunities = **40**
- Average deal value = **15000**
- Win rate = **25%**
- Sales cycle length = **30**

Then:

- Sales velocity = **40 × 15000 × 0.25 ÷ 30**
- Sales velocity = **5000.00**

## What is sales velocity?

Sales velocity measures how quickly revenue moves through the sales process.

## When to use this calculator

Use this calculator when you want to:

- review pipeline health

## Common mistakes

Common mistakes include:

- using outdated opportunity counts
- overstating win rate
- ignoring long sales cycles
- mixing pipeline stages with very different close probabilities

## Sales velocity vs revenue forecast

These are closely related.

- **Sales velocity** estimates how quickly pipeline value moves
- **Revenue forecast** projects future revenue based on broader assumptions
